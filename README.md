# dingRobot

#### 介绍
钉钉机器人消息推送 简单封装

#### 安装教程

1. 在`composer.json`文件中`repositories`部分加入以下配置

* `gitHub`

```json
{
    "type": "git",
    "url": "https://github.com/chanlly/dingRobot.git"
}
```

* `gitee`

```json
{
    "type": "git",
    "url": "git@gitee.com:canl/dingRobot.git"
}
```

2. 执行`composer require "chanlly/dingRobot"`
